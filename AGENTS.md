# Skills — Cardapio

Spring Boot REST API — Cardápio de pizzaria com autenticação JWT.

## Skills disponíveis (globais em ~/.agents/skills/)

### Planejamento
- `brainstorming` — antes de implementar novas features
- `writing-plans` — planejar tarefas multi-passo

### Desenvolvimento
- `tdd` — red-green-refactor para services e controllers

### Depuração
- `systematic-debugging` — ao encontrar bugs (ex: N+1 queries, JWT filter)
- `diagnose` — debugging estruturado de problemas complexos

### Revisão
- `requesting-code-review` — antes de finalizar implementações
- `receiving-code-review` — ao receber feedback
- `verification-before-completion` — verificar antes de afirmar conclusão

### Segurança
- (aplicar boas práticas: JWT, CORS, validação de input)

### Comunicação
- `caveman` — respostas ultra-compactas (economia de tokens)
- `zoom-out` — visão geral da arquitetura

## Fluxo recomendado
1. `brainstorming` → entender requisitos
2. `writing-plans` → plano de ação
3. `tdd` → implementar com testes
4. `verification-before-completion` → verificar
5. `requesting-code-review` → revisão final
